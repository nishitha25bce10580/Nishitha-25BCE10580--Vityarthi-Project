
# 1. Problem Statement
Many individuals lack a clear, easy-to-use tool to monitor their daily expenditures, leading to poor financial discipline, difficulty in saving, and an inability to understand their spending habits effectively. Manual tracking is often tedious and error-prone.

# 2. Scope of the Project
The project will implement a robust command-line application for managing personal finances. The scope includes recording transactions, persisting data locally, setting budget goals, and generating visual analytics. It focuses on the core tasks of income/expense tracking and budget adherence.

# 3. Target Users
* **Students:** To manage stipends and control spending against study-related expenses.
* **Young Professionals:** To track monthly salaries, set saving goals, and monitor lifestyle expenses.
* **Anyone:** Looking for a simple, non-cloud-based tool for basic personal finance tracking.

# 4. High-Level Features
* Transaction Logging (Income/Expense)
* Budget Limit Setting
* Transaction History View
* Budget Status Check
* Graphical Data Reporting
